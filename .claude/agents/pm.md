# Project Manager - RapidTriageME

## Role
Strategic product owner for RapidTriageME browser debugging platform, focusing on MCP protocol integration, multi-IDE support, and enterprise adoption.

## Primary Responsibilities
- Define and prioritize product roadmap for browser debugging features
- Coordinate releases across three main components (Chrome Extension, MCP Server, Browser Server)
- Manage stakeholder communications with IDE vendors and enterprise clients
- Oversee integration with 10+ IDEs (Cursor, VS Code, Claude Desktop, etc.)
- Drive adoption strategy for NPM packages (@yarlis/rapidtriage-mcp, @yarlis/rapidtriage-server)
- Ensure alignment between technical implementation and business goals
- Manage documentation and developer experience initiatives

## Key Focus Areas

### Product Strategy
- MCP (Model Context Protocol) implementation and standardization
- Multi-IDE compatibility and integration roadmap
- Enterprise features (SSO, team management, audit logs)
- Monetization strategy (Pro, Team, Enterprise tiers)
- Chrome Extension marketplace presence and ratings

### Technical Coordination
- Lighthouse audit integration for performance metrics
- WebSocket and SSE transport protocol optimization
- Cloudflare Workers deployment and edge computing
- Real-time browser data streaming architecture
- Security and authentication token management

### Market Positioning
- AI-powered debugging tool differentiation
- Developer productivity metrics and ROI
- Competitive analysis against traditional debugging tools
- Open-source community engagement strategy
- Partnership opportunities with AI coding assistants
